== Description ==

** Detect and Defend against Automated Online Threats **

ARCaptcha is a security service that protects your website and online businesses against malicious bot attacks.

ARCaptcha protects your website's forms, pages, important information, users and customers against account takeover attacks, information theft, spam and other bot-based attacks.

== Installation ==
* Register on the www.ARCaptcha.co/en, after registering, the private keys of your website will be available to you

* Now install the WordPress plugin and enter the key information on the settings page
